# Instructor Notes: Lab 5

## Primary Goals

Lab 5 introduces two major ideas:

1. Object-oriented programming with classes and objects
2. Git feature branches

The Git goal should remain simple: branch, commit, merge. Avoid merge conflicts, rebasing, or remote branch complications.

## Suggested Live Modifications

- Add `number_of_scores()`
- Change the grading scale
- Add one new student
- Modify `__str__`
- Reject invalid scores with a printed warning

## Important Discussion Points

- Objects combine data and behavior.
- `self` refers to the current object.
- The constructor initializes the object's state.
- `main` should remain stable; feature branches are for development.
