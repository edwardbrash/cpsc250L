# Instructor Notes: Lab 6

## Primary Goals

Lab 6 moves from one object to many objects.

Students should begin seeing objects as structured records that also know how to compute things about themselves.

## Suggested Live Modifications

- Print only A students
- Count passing students
- Add a new student to the CSV file
- Sort not required yet, but can be discussed
- Add a `is_passing()` method

## Watch For

Students may confuse:
- a class with an object
- an object with a dictionary
- a list of scores with a list of students
